# Gen-AI Usage Notes

## Principle

Gen-AI is used as a learning, planning, documentation, and troubleshooting assistant.

It is not used as a replacement for understanding the official documentation or blindly generating production code.

## Allowed Usage

Gen-AI may be used for:

- Understanding concepts
- Planning project structure
- Brainstorming architecture
- Reviewing documentation
- Creating implementation checklist
- Troubleshooting general errors
- Improving explanation and reflection notes

## Avoided Usage

Gen-AI should not be used to:

- Autogenerate the entire project without understanding
- Replace official documentation
- Copy code blindly
- Skip manual testing
- Hide implementation decisions

## Official Documentation Priority

Official documentation should remain the primary technical reference for:

- React
- ASP.NET Core
- Entity Framework Core
- MSSQL
- JWT authentication
