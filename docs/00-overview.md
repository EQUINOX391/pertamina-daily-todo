# Project Overview

## Project Name

Pertamina Daily TODO List

## Description

A fullstack Daily TODO List application with authentication and user-specific TODO management.

## Study Case Background

The challenge requires building a CRUD Daily TODO List fullstack application using React JS for frontend and .NET with Entity Framework for backend. MSSQL is used as the database.

The expected deadline is Friday, 15 May 2026.

The required features are:

- Register
- Login
- JWT stateless authentication
- Create TODO
- Read TODO
- Update TODO
- Delete TODO

The project is expected to show not only working functionality, but also the developer's approach in designing fullstack architecture, integrating frontend and backend through API, and applying clean code, validation, error handling, separation of concern, and security awareness.

## Main Objective

Build a simple, clean, and documented fullstack application that demonstrates practical understanding of:

- React frontend development
- ASP.NET Core Web API backend development
- Entity Framework Core ORM usage
- MSSQL database integration
- JWT stateless authentication
- CRUD API design
- Clean code and separation of concerns

## Personal Development Intention

This project is intentionally developed as a professional study case, not merely a simple CRUD application.

The development process should demonstrate:

- Structured planning
- Clear documentation
- Incremental progress
- Security awareness
- Testing notes
- Decision log
- Reflection log
- Realistic execution within the given deadline
